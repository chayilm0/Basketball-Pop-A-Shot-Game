# Basketball-Pop-A-Shot-Game
A fun, interactive basketball shooting game built with HTML and JavaScript using the Canvas API. Players aim with their mouse and shoot balls into a basket by clicking down, scoring 2 points for each made basket. The game includes a bonus round where scoring is worth 3 points each.

## Features

- Dynamic player and projectile mechanics
- Randomizer for ball position after each shot
- Gravity and physics for realistic ball motion
- Particle effects for scoring shots
- Timer-based gameplay with standard and bonus rounds
- Score tracking and modal pop-ups for end-of-game display

## How to Play

1. Click **Start Game** to begin.
2. Click on the canvas to shoot the ball toward the basket.
3. Score 2 points per successful basket. Can only score if the ball goes above the rim (red rectangle) and hits the top in a descending motion near its center.
4. If you reach a score threshold of 12, before the initial timer reaches 0, the **Bonus Round** becomes available:
    - Scoring increases to 3 points per basket
    - Timer is set to 10 seconds
5. The game ends when the timer reaches 0 again.

## Controls

- **Click**: Shoot the ball.
- **Moving Mouse**: Aim where you are shooting based on the position of the ball.
- **Start Game / Start Bonus**: Begin a round.
- **Modal**: Displays score at the end of each round.

## Running the Program
Intended to be run using Live Server.
